### 块操作
| 快捷键名称 | 命令 | 快捷键  |
| ------------ | ------------ | ------------ |
|C-x r k ||剪切一个矩形块|
|C-x r y ||粘贴一个矩形块|
|C-x r o ||插入一个矩形块|
|C-x r c ||清除一个矩形块(使其变成空白)|
|C-x r t ||在选定区域的所有列前插入样的字符|





### buffer 相关
| 快捷键名称 | 命令 | 快捷键  |
| ------------ | ------------ | ------------ |
| C-x C-q |read-only-mode|emacs解除read-only的buffer锁定|
